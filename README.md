# Make Circles

Take a csv input and output a kml file with radius circles plotted around each point

The .csv input should have at least the following columns:
- latitude
- longitude
- accuracy_radius

## Usage
```bash
$ python3 make_circles.py -i input.csv -o output.kml
```
